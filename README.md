**AI Resume Screening System**

This project is a simple AI-based Resume Screening System built as part of a GenAI internship task.
It analyzes resumes based on a given job description and evaluates candidates accordingly.

🔹 Extracts relevant skills from resumes
🔹 Matches them with job requirements
🔹 Assigns a score (0–100)
🔹 Provides explanation for evaluation

📊 Pipeline:
Resume → Skill Extraction → Matching → Scoring → Explanation
